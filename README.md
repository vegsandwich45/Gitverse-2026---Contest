# 🚀 GITVERSE 2026 – Hands-on Git Challenge

Welcome to the **GITVERSE 2026 Contest**!  
This challenge is designed to help you apply Git concepts through real-world scenarios.

---

## 🎯 Objective

Use the Git knowledge from the workshop to solve practical problems involving branching, merging, and collaboration.

---

## 👤 Participation

- This is an **individual challenge**
- Each participant must complete and submit their own work

---

## 📌 Instructions

1. Fork this repository
2. Clone your fork to your local machine
3. Navigate into the project directory

---

## 🧩 Tasks

- Each question is provided in a separate file
- For each question:
  - Create the branch mentioned in the file
  - Write your answer (**Git commands only**) below the problem
  - Commit the file in the same branch

---

## ⚠️ Rules

- Write **only Git commands** (no explanations)
- Use **one command per line**
- Do NOT delete or modify the question
- Follow the branch naming exactly as given
- Ensure your commands are logically correct

---

## 🔄 Final Submission

1. Push all your branches to your forked repository
2. Create a **Pull Request (PR)** to the original repository

---

## 💡 Tips

- Think step-by-step before writing commands  
- Focus on the workflow you practiced during the session  
- Stay calm and approach each problem logically  

---

## 🚀 Good Luck!

Now it’s your turn to work like a real developer. 💻🔥
